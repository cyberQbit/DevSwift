### Node Paket Yöneticileri
```
npm (Node.js ile varsayılan gelir)

PNPM: npm install -g pnpm
```

### Python Paketleri
```
pip (Python ile varsayılan gelir)
Popüler paketler:
  - requests: pip install requests
  - pandas: pip install pandas
  - tensorflow: pip install tensorflow
  - flask: pip install flask
  - django: pip install django
```


## 🔄 Yeni Araç Eklemek

DevSwift'e yeni araç eklemek için:

### 1. Winget ID Bulma
```powershell
winget search "araç adı"
```

### 2. DevSwift.bat'ta Fonksiyon Ekleme
```batch
:InstallYeniArac
echo [Araç Adı] Kurulu icerik kontrol ediliyor...
winget install -e --id Yayinci.Aracadi --silent --accept-package-agreements --accept-source-agreements 2>nul
if %errorLevel% equ 0 (
    call :LogMessage "Araç kuruldu"
    echo [OK] Araç kuruldu
) else (
    call :LogMessage "Araç kurulumu basarisiz"
)
echo.
goto :EOF
```

### 3. Menüye Ekleme
```batch
if "%%i"=="NN" call :InstallYeniArac
```
